What is JavaScript?

JavaScript is the programming language that adds interactivity and custom behaviours to a web site. With JavaScript you can create interactive buttons, search bars or even to create a website itself.

Script Tag

JavaScript can be added to a web page by using a script tag (<script>). There's still 2 ways to implement this tag, you can either have embedded script where you write the JS code within the script tags or with an external script which you can reference in the opening script tag using the src attribute followed by the URL of the script file. One way you can use JavaScript is to make a list.
