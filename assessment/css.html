What is CSS?

Cascading Style Sheets (CSS) is a language used to define the presentation of documents written in HTML. CSS is used to choose the way the document is displayed to the user. This is a quote taken from 'Learning Web Design: a beginner's guide to HTML, CSS, JavaScript, and web graphics'.

Selectors

Selectors are what CSS uses to reference the tag it is defining and can be the tag itself, the class attribute or the id attribute. When defining the tag the selector only needs to be the name of the tag. When defining a class the selector is .class where class is the name of the class. When defining an id the selector is #id where id is the name of the id. There are other selectors you can use, and they can be found here.

Declaration

A declaration in CSS is the indicator of what aspect of the element you are changing such as the colour or width etc. Declarations are made up of both a property and a value, so if you were changing the colour of a element the property would be color and the value would be the colour your changing it to for example blue. There are other declaration you can use, and they can be found here.
