What is HTML?

HTML is not a programming language, it's a markup language. HTML is the language used to create a web page. HTML is used for identifying and describing the components of a document for example headings, paragraphs, lists, images etc.

Tags

To write a document in HTML you need to use tags. A tag is encased within 2 inequality signs (< >), and has a closing tag which has a forward slash (< / >) after the first inequality sign. In HTML there is 3 main 'parent' tags which are html, head and body. The html tag is the parent tag for head and body. The head tag is for metadata such as the title, the link to the CSS file etc. The body tag is for what you want to display on the webpage. Within the head and body tag there is multiple tags you can use and you can find them all here.

Attributes

Attributes are used in HTML to provide additional information about the html tag. Some attributes are used to create hyperlinks using the href attribute. You can also use attributes to create a class or id, these are used to make it easier to edit the tags using CSS. A class is used to group multiple tags into one for CSS to edit whereas the id attribute is for one tag and makes it so CSS can directly edit it. There is many more attributes that you can use and you can find them all here.
